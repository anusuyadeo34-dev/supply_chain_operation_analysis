## 📌 Project Overview

The **APL Profitability Dashboard** is an interactive Streamlit-based data analytics dashboard developed for analyzing customer profitability, product performance, delivery risks, discount impact, and regional sales insights in supply chain operations.

This dashboard helps users:
- Track revenue and profit
- Analyze customer and product performance
- Identify delivery risks
- Understand discount impact on profits
- Explore regional profitability trends
- Download filtered datasets
